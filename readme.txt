RectangleRun is a simple game made in python for  beginners for learning.

Purpose

This game is intended for beginners who want to:

    1.learn Python by building something real.

    2.understand basic game logic including like,score system , basic menu logic , collision using AABB,increasing difficulty.

    3.explore game development using raylib.

 Technical Details

    1.AABB rectangular collision

    2.A really basic option to change difficulty levels

    3.Time-based scoring system
    
    4. Increasing difficulty with time.

 How to Play

    1.W / A / S / D — Move the player (blue rectangle)

    2.Avoid the white enemy rectangle

    3.R — Restart after collision

    4.ESC — Close the game

The goal is simple: survive as long as possible.
